В Java Reflection позволяет нам изучать классы,
интерфейсы, конструкторы, методы, поля
и управлять ими во время выполнения.

В Java есть класс с именем Class, экземпляры которого хранят
всю информацию об объектах и классах во время выполнения.
Объект Class использоваться для Reflection.


Чтобы во время выполнения программы изучить класс Java,
нам сначала нужно получить доступ к объекту класса Class.

Используя объект класса Class, мы cможем вызывать различные
методы для получения информации о методах, полях и конструкторах,
присутствующих в изучаемом нами классе.


Существует три способа получить ссылку на объекты класса Class:

1. Используя метод forName()

class Dog {...}

// create object of Class
// to reflect the Dog class
Class a = Class.forName("Dog");

Здесь метод forName() принимает полное имя класса (вместе с пакетом в котором
находится данный класс), которое будет отражено в качестве своего аргумента.


2. Используя метод getClass()

// create an object of Dog class
Dog d1 = new Dog();

// create an object of Class
// to reflect Dog
Class b = d1.getClass();


Здесь мы используем объект класса Dog для получения ссылки на объект класса Class.



3. Используя расширение .class

// create an object of Class
// to reflect the Dog class
Class c = Dog.class;


Напишите программу в которой продемонстрируйте использование
всех трёх способов получения ссылки на объект класса Class.
И убедитесь, что все ссылки полученные каждым из перечисленных
выше способов в действительности указывают на один и тот же объект
(тоесть равны друг другу ==).
